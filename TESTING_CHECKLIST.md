# ✅ รายการตรวจสอบก่อนใช้งานจริง

## 🎯 ภาพรวม
ระบบสถานะอสังหาริมทรัพย์ถูกปรับปรุงแล้ว ก่อนให้ผู้ใช้งานเข้าใช้จริง กรุณาตรวจสอบตามรายการนี้

---

## 1️⃣ การตรวจสอบพื้นฐาน

### ✅ หน้ารายการทรัพย์ (`/properties`)
```
URL: https://www.mesuk.co.th/agent/properties
```

- [ ] หน้าโหลดขึ้นมาสมบูรณ์ ไม่มี error
- [ ] แสดงสถิติ 3 การ์ดด้านบน: ทรัพย์ทั้งหมด / ไม่ว่าง / ว่าง
- [ ] การ์ด "ไม่ว่าง" แสดงเป็น**สีแดง** ✅
- [ ] การ์ด "ว่าง" แสดงเป็น**สีเขียว** ✅
- [ ] ตารางแสดงรายการทรัพย์ทั้งหมด
- [ ] แถบสีด้านซ้ายของแต่ละแถวแสดงสีตามสถานะ:
  - [ ] ไม่ว่าง (เช่าอยู่/ยืนยันแล้ว) = แดง
  - [ ] โครงการในอนาคต (ยืนยันมัดจำ) = น้ำเงิน
  - [ ] จองแล้ว = เหลือง
  - [ ] ว่าง = เขียว (หรือเทาอ่อน)

### ✅ หน้าภาพรวม (`/rental-rates`)
```
URL: https://www.mesuk.co.th/agent/rental-rates
```

- [ ] หน้าโหลดขึ้นมาสมบูรณ์
- [ ] สถิติด้านบนแสดงถูกต้อง
- [ ] กราฟแสดงถูกต้อง
- [ ] ตารางแสดง Status Badge สีถูกต้อง:
  - [ ] ว่าง = เขียว
  - [ ] ไม่ว่าง = แดง

### ✅ หน้ารายละเอียดทรัพย์ (`/properties/{id}`)
```
URL: https://www.mesuk.co.th/agent/properties/345
```

- [ ] หน้าโหลดขึ้นมาสมบูรณ์
- [ ] ข้อมูลทรัพย์แสดงถูกต้อง
- [ ] ข้อมูลผู้เช่าแสดงถูกต้อง
- [ ] ข้อมูลการชำระเงินแสดงถูกต้อง

## 2️⃣ การทำงานของระบบ

### ✅ การค้นหา
- [ ] พิมพ์ชื่อทรัพย์ → แสดงผลลัพธ์ถูกต้อง
- [ ] พิมพ์รหัสทรัพย์ → แสดงผลลัพธ์ถูกต้อง
- [ ] พิมพ์ชื่อผู้เช่า → แสดงผลลัพธ์ถูกต้อง
- [ ] พิมพ์เบอร์โทร → แสดงผลลัพธ์ถูกต้อง
- [ ] คลิกปุ่ม X → ล้างการค้นหา

### ✅ การกรอง
- [ ] คลิก "ทั้งหมด" → แสดงทรัพย์ทั้งหมด
- [ ] คลิก "ไม่ว่าง" → แสดงเฉพาะทรัพย์ที่ไม่ว่าง
- [ ] คลิก "ว่าง" → แสดงเฉพาะทรัพย์ที่ว่าง

### ✅ การคลิก
- [ ] คลิกแถวทรัพย์ → เข้าหน้ารายละเอียด
- [ ] ไม่มีแถวว่างคลิกไปไหนได้

### ✅ การแสดงผล
- [ ] รูปผู้เช่าแสดงถูกต้อง
- [ ] ชื่อผู้เช่าแสดงถูกต้อง
- [ ] เบอร์โทรแสดงถูกต้อง
- [ ] ค่าเช่าแสดงถูกต้อง (มีจุลภาค)
- [ ] วันที่เช่าแสดงเป็นภาษาไทย
